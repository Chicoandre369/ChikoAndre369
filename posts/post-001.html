Después de evaluar varias opciones para reforzar mi perfil técnico, decidí enfocarme en dos certificaciones de Google que se complementan bien entre sí: Google Cybersecurity y Google IT Automation with Python.

¿Por qué estas dos?

En vez de ir por Data Analytics (la opción más popular), elegí esta combinación porque me da una base sólida en seguridad y automatización, dos áreas con alta demanda y que se conectan directamente con lo que ya estoy construyendo en Gagdetop (mi proyecto de automatización con Python).

1. Google Cybersecurity

  • Fundamentos de seguridad informática
  • Gestión de riesgos y cumplimiento normativo
  • Redes y seguridad de redes
  • Linux y SQL aplicados a seguridad
  • Detección y respuesta ante incidentes
  • Python aplicado a automatización de tareas de seguridad

2. Google IT Automation with Python

  • Fundamentos de Python
  • Uso de Git y GitHub (control de versiones)
  • Automatización con scripts
  • Resolución de problemas técnicos con código
  • Introducción a control de configuración y sistemas en la nube

Estado actual

Por ahora estoy en la etapa de planificación y organizando el orden de estudio. La idea es avanzar en paralelo con mis ramos de AIEP (Bases de Datos, Matemática) y usar Gagdetop como laboratorio práctico para aplicar lo que vaya aprendiendo en automatización con Python.
